## P1 — Native ownership and completed-media durability

- [x] **P1.1 / AGG1-1 (High/High): isolate abandoned GL instances.** Make one `GlPipeline`
  instance own exactly one native generation after a stop timeout. A timed-out instance becomes
  permanently retired and cannot restart itself. Let `CameraEngine` atomically replace it with a
  fresh instance, make `RendererAssists` resolve the current instance, and identity-gate every
  engine callback that a retired instance can deliver (`onInputReady`, preview health, analysis,
  Camera2 input ownership, encoder attach/detach, and release completion). The old handler may later unwind, but it can touch
  only its old object's EGL/renderer/surface fields. Every start/bind/encoder/stop transaction must
  capture one `ownedGl`; `stop` requires both thread exit and checked native release for `STOPPED`, and an `ABANDONED` result
  compare-and-swaps that owner to a fresh object without waiting for `onResourcesReleased` (which
  correctly never fires when native release is unsafe). Add an Android-free atomic owner-slot seam
  and race tests proving A cannot read, mutate, stop, or publish B state.
- [x] **P1.2 / AGG1-4 (Medium/High): make finalized-media evidence non-optional.** Replace ignored
  `markWriteComplete` calls with a required result and bounded marker retry. Add a structural HEIF/
  ISO-BMFF completion probe that validates top-level box extents and required container boxes
  without decoding a full-resolution bitmap; recovery may adopt only a structurally complete file.
  Add the failed-COMPLETE + failed-publish + restart matrix and assert every row reaches adopt,
  proven-incomplete delete, or explicit retained/error state—never silent indefinite pending.
- [x] **P1.3 / AGG1-8 (Medium/Medium): validate tolerated muxer-stop output.** If the one
  sample-less degraded-audio `MediaMuxer.stop()` exception is tolerated, close native owners and
  require a `MediaExtractor`-readable video track before COMPLETE/publication. Extend the pure
  publish policy so failed or skipped validation cannot surface a clip.
- [x] **P1.4 / AGG1-11 (Medium/High): return observable recovery outcomes.** Have orphan cleanup
  return a sanitized `RecoveryReport` with scanned/adopted/deleted/retained/error counts and failure
  classes. Continue one collection when the other fails, emit one low-volume debug transition, and
  use a bounded launch retry for provider errors. Add query-error, mixed-collection, publish-failure,
  and exhausted-retry tests.
